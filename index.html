常見問題 ＆ 操作說明
🔗 第一次使用：怎麼配對？
1. 一方進入「設定 → 個人資料 → 配對資訊」產生邀請碼
2. 把邀請碼分享給另一半
3. 另一半在同頁面輸入邀請碼完成配對
4. 配對成功後所有功能即時共享
📝 清單怎麼用？
1. 清單頁點「＋」新增想做的事，選擇分類
2. 勾選 ✅ 完成後可填寫心得
3. 完成的項目才可以關聯到回憶錄
💡 「幫我決定」可從未完成清單隨機抽一個
💡 靈感題庫怎麼用？
清單靈感：清單頁右上角「💡靈感」→ 選清單靈感 → 點「加入清單」
挑戰靈感：挑戰頁右上角「💡靈感」→ 選挑戰靈感 → 點「加入挑戰」→ 表單自動帶入名稱，填截止日期後新增
自訂靈感：設定 → 管理員專區 → 靈感題庫管理 → 點「匯入靈感」
匯入格式（JSON 陣列）：
清單：[{"n":"項目名稱","c":"想去哪"}]
挑戰：[{"n":"挑戰名稱","c":"探索類"}]
清單分類：想去哪／想吃什麼／想做什麼／想看什麼
挑戰分類：探索類／飲食類／運動類／創作類／感情類／娛樂類／學習類／其他
⭐ 挑戰怎麼用？
1. 挑戰頁點「＋」新增挑戰，填入名稱和截止日期
2. 兩人都需勾選才算完成（進度條顯示 0/2、1/2、2/2）
3. 已完成的挑戰才可以關聯到回憶錄
💡 截止日過了會自動移至「已失敗」；誤觸可在已完成頁「恢復進行中」（截止日未過才能恢復）
📷 回憶錄怎麼建立？
1. 先把相關清單打勾完成、挑戰標記完成
2. 回憶錄頁點「＋」新增，填入名稱、日期、地點
3. 點「選擇清單或挑戰」→ 系統自動顯示日期範圍（前後5天）內已完成的項目
4. 點「選擇關聯餐廳」→ 連結那次去的餐廳
5. 建立後可在詳情頁「新增照片」上傳照片
💡 超出日期範圍的項目點「顯示其他已完成」展開
🍽️ 餐廳雙人評分怎麼用？
1. 新增餐廳時先填自己的評分（1-5星）
2. 對方收到通知 → 進入餐廳頁 → 點「詳情」→ 選星星 → 儲存評分
3. 列表顯示兩人平均分（支援半顆星）
4. 還沒評分的餐廳會顯示「待評」標籤，頁面頂部橫幅顯示待評數量
💡 每人可隨時點「詳情」修改自己的評分
🔔 通知怎麼運作？
APP 通知：所有動態自動記錄在通知頁（🔔），有未讀時顯示紅點
提醒通知（暖黃色）：紀念日、生日、挑戰到期、行事曆 → 在「設定 → 通知偏好」設定提前幾天
Discord 通知：設定 → Discord 通知設定，貼上 Webhook URL 即可
💡 清除通知只影響自己，不影響對方的通知頁
📸 照片存在哪裡？
照片存在 Cloudinary 雲端，免費方案有 25 GB（約 5,000～12,000 張）。
設定方式：設定 → Cloudinary 照片設定 → 填入 Cloud Name 和 Upload Preset
查看使用量：登入 cloudinary.com → Home → Go to Usage Reports → 看 Storage
🎨 主題色系和 APP 名稱可以改嗎？
主題色系：設定頁「外觀」各自設定，不影響對方
APP 名稱：管理員設定 → APP 名稱設定（兩人即時同步）
💡 主畫面捷徑名稱不會自動更新，需刪除後重新加入主畫面
🔒 忘記隱私鎖 PIN 怎麼辦？
鎖定畫面點「忘記 PIN？」→ 送出請求 → 等另一半在通知頁同意 → 重新設定
💾 可以備份資料嗎？
設定 → 管理員專區 → 匯出資料 → 下載 JSON 檔案
如需還原：管理員專區 → 還原資料 → 上傳 JSON 檔案

Discord 通知設定
🔔 什麼是 Discord 通知？
設定後，APP 裡的操作（新增清單、完成挑戰、新增回憶錄等）會同步發送通知到你們指定的 Discord 頻道，就算沒開 APP 也能看到對方的動態。
📋 設定步驟
1在 Discord 建立一個你們兩人的共用伺服器（或使用現有的）
2選一個頻道（如 #通知），點頻道名稱旁的 ⚙️ 編輯頻道
3點左側 整合 → Webhooks → 建立 Webhook
4點 複製 Webhook URL，貼到下方欄位，按儲存
5按 🔔 測試 確認頻道有收到訊息
Cloudinary 照片設定
什麼是 Cloudinary？
照片存在 Cloudinary 雲端，不佔用 Firebase 空間。免費方案有 25 GB 儲存空間（約 5,000～12,000 張），上傳後會自動壓縮。
設定步驟（第一次使用）
1 前往 cloudinary.com 免費註冊帳號
2 登入後，右上角 Dashboard 可看到 Cloud Name（一串英文字）
3 點上方 Settings → Upload → Upload presets，新增 Preset，模式設為 Unsigned，記下名稱
4 把 Cloud Name 和 Preset Name 填入下方，儲存即可
查看儲存使用量
登入 Cloudinary → 左側點 Home → 點 Go to Usage Reports → 看 Storage 欄位的 MB 數字
